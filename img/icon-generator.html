üé® Alternative Icon Creation Methods

If you prefer to create custom icons, here are the recommended specifications:

App Store Icon (512x512):

  • Format: PNG with transparency
  • Background: Gradient from #667eea to #764ba2
  • Border radius: 80px (15.6%)
  • Main symbol: üì± or custom app icon
  • Text: "CORDOVA" (48px, bold) + "APP GENERATOR" (28px, normal)

iOS App Icon (114x114):

  • Format: PNG with transparency
  • Background: Same gradient as above
  • Border radius: 18px (15.8%)
  • Simplified design with "CAG" text or just the üì± symbol

Promotional Banners (1280x800):

  • Format: PNG or JPG
  • Background: Same gradient with subtle pattern
  • Layout: Text on left, visual element on right
  • Typography: Large title (72px), subtitle (28px), features (18px)

Color Palette:

  • Primary: #667eea (Blue)
  • Secondary: #764ba2 (Purple)
  • Text: White (#ffffff)
  • Accent: rgba(255,255,255,0.1) for patterns
